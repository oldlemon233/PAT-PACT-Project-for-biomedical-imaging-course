# 数据目录 (Data Directory)

本目录用于存储项目中使用的所有数据。

## 子目录说明

### raw/
存储原始数据，包括：
- 原始PACT信号数据
- 实验采集的数据
- 外部数据源

### processed/
存储经过预处理后的数据：
- 归一化后的数据
- 数据增强后的结果
- 清洗后的数据集

### gold_standard/
存储金标准数据：
- 仿真生成的真实标注
- 专家标注的ground truth
- 参考标准数据

### simulation/
存储数字仿真生成的数据：
- 仿真生成的PACT信号
- 仿真参数记录
- 仿真中间结果

## 注意事项

- 大型数据文件建议不要直接提交到Git，使用Git LFS或外部存储
- 每个数据集应包含对应的元数据描述文件（JSON或YAML格式）
- 保持数据版本管理，记录数据来源和处理流程
