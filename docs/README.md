# 文档目录 (Documentation)

本目录用于存储项目相关的所有文档。

## 子目录说明

### reports/
项目报告：
- 阶段性报告
- 实验报告
- 最终项目报告
- 技术文档

### presentations/
演示文稿：
- 项目PPT
- 阶段汇报slides
- 答辩演示文稿
- 海报设计

### papers/
相关论文和参考资料：
- 参考文献
- 综述文章
- 相关研究论文
- 理论背景资料

## 注意事项

- 使用Markdown或LaTeX编写文档，便于版本控制
- PPT建议使用Google Slides或在线工具，导出PDF版本到此目录
- 保持文档命名规范：日期_类型_版本.ext
