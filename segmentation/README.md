# 分割模块 (Segmentation Module)

本目录包含半暗带分割网络相关的代码和资源。对应**任务2.2：分割网络**。

## 子目录说明

### models/
分割网络模型定义：
- U-Net及其变体
- 语义分割网络
- 自定义分割架构
- 损失函数（Dice Loss, Focal Loss等）

### scripts/
训练和推理脚本：
- 分割网络训练脚本
- 验证和测试脚本
- 推理脚本
- 后处理脚本

### configs/
配置文件：
- 网络架构配置
- 训练超参数
- 数据增强策略
- 类别权重配置

### checkpoints/
模型权重：
- 训练检查点
- 最佳性能模型
- 不同阶段的模型快照
- 模型集成备份

### results/
分割结果：
- 分割掩码
- 评估指标（Dice, IoU等）
- 可视化对比
- 错误案例分析

## 使用说明

1. 确保重建模块已生成重建图像
2. 在`configs/`中配置分割网络参数
3. 运行`scripts/train.py`训练分割模型
4. 使用`scripts/inference.py`进行半暗带分割
5. 结果和评估指标保存到`results/`
