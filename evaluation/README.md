# 评估模块 (Evaluation Module)

本目录用于系统整合与量化分析。对应**任务3：系统整合与量化分析**。

## 子目录说明

### metrics/
评估指标实现：
- Dice系数
- IoU (Intersection over Union)
- 灵敏度和特异性
- Hausdorff距离
- 体积测量误差
- 自定义评估指标

### scripts/
评估流水线脚本：
- 端到端评估流水线
- 批量评估脚本
- 结果统计分析
- 对比实验脚本

### results/
评估结果：
- 量化评估报告
- 统计分析结果
- 性能对比表格
- 消融实验结果

### visualizations/
可视化结果：
- 分割结果可视化
- 性能曲线图
- 对比图表
- 错误分布图

## 使用说明

1. 配置评估流水线参数
2. 运行端到端评估脚本
3. 自动调用重建和分割模块
4. 与金标准对比计算评估指标
5. 生成可视化结果和报告
