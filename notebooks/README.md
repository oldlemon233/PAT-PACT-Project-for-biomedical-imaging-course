# Notebooks 目录

本目录用于存储Jupyter notebooks，用于快速实验、数据探索和结果可视化。

## 使用场景

- 数据探索和可视化
- 算法原型开发
- 实验记录和分析
- 教学演示

## 命名规范

建议使用编号和描述性名称：
- `01_data_exploration.ipynb` - 数据探索
- `02_simulation_demo.ipynb` - 仿真演示
- `03_reconstruction_experiment.ipynb` - 重建实验
- `04_segmentation_analysis.ipynb` - 分割分析
- `05_results_visualization.ipynb` - 结果可视化

## 注意事项

- Notebooks应包含清晰的说明和注释
- 运行前确保环境配置正确
- 大规模实验建议使用scripts/而不是notebooks
- 定期清理notebook输出以减小文件大小
