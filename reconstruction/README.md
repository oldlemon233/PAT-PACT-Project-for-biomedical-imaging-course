# 重建模块 (Reconstruction Module)

本目录包含图像重建网络相关的代码和资源。对应**任务2.1：重建网络**。

## 子目录说明

### models/
重建网络模型定义：
- 深度学习重建网络架构
- 自定义网络层
- 损失函数定义
- 网络变体

### scripts/
训练和推理脚本：
- 训练脚本
- 验证脚本
- 推理脚本
- 数据加载器

### configs/
训练配置文件：
- 网络超参数
- 训练策略配置
- 数据集配置
- 优化器参数

### checkpoints/
模型检查点：
- 训练过程中的模型权重
- 最佳模型
- 预训练模型
- 模型版本记录

### results/
重建结果：
- 重建后的图像
- 评估指标
- 训练日志
- 可视化结果

## 使用说明

1. 在`configs/`中配置网络和训练参数
2. 使用`scripts/train.py`进行模型训练
3. 训练过程中模型保存到`checkpoints/`
4. 使用`scripts/inference.py`进行图像重建
5. 结果保存到`results/`目录
