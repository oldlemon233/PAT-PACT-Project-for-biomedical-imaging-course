# 仿真模块 (Simulation Module)

本目录包含数字仿真相关的所有代码和配置。对应**任务1：构建数字仿真数据集**。

## 子目录说明

### models/
仿真模型定义：
- 简化的组织模型
- 光学传播模型
- 声学传播模型
- 缺血性脑卒中半暗带模型

### scripts/
仿真执行脚本：
- 生成金标准的脚本
- 模拟PACT信号的脚本
- 批量仿真处理脚本
- 数据验证脚本

### configs/
仿真配置文件：
- 仿真参数配置（JSON/YAML）
- 组织光学参数
- 几何模型参数
- 探测器配置

### results/
仿真结果输出：
- 仿真生成的数据集
- 仿真日志
- 中间结果
- 质量评估报告

## 使用说明

1. 在`configs/`中配置仿真参数
2. 运行`scripts/`中的脚本执行仿真
3. 结果自动保存到`results/`目录
4. 金标准数据应复制到`../data/gold_standard/`目录
